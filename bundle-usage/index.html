Developer Bundle

How to use the developer bundle

The developer bundle has four purposes;

  1. To provide sample applications that can be used to explore how AppFlow works
  2. To provide reference applications to test integration of your own application with in a flow
  3. To provide code examples of how to interact with the APIs
  4. To provide an AppFlow Configuration Provider (named AppFlow Settings) tailored for the above use cases

AppFlow Settings App

The AppFlow Settings application (green icon on launcher) provides a way to manage the flow configurations on the device, as well as the general AppFlow and FPS settings.

This application comes with a set of pre-defined flows to go with the samples, but in addition, it will scan for AppFlow compliant flow services and automatically add them to the relevant flows and stages.

Limitations

As there is a pre-defined set of flows defined in this bundle, it is possible that it does not provide all the functions that your application requires. For these cases, contact AEVI to discuss options.
