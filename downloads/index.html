AppFlow Downloads

There are a few different components relevant for anyone who wants to test AppFlow or start integrating with it;

  • Flow Processing Service (FPS) - handles initiation requests and processes the flows
  • Developer Config Provider - provides flow configurations and settings to AppFlow services and FPS
  • API Samples - for testing and validating integrations

If any of these components changes, a new developer bundle will be created and published to this page for downloads. In the table below you can see what versions of each component is and what API version they are integrated with.

The first official bundle with AppFlow v2 is v6 (v1 - v5 were internal releases).

Note that these components require Android API level 21+ (v5.0, Lollipop and above).
