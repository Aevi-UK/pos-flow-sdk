What is AppFlow?

AppFlow provides a set of APIs that allow simple and straight forward integration into any point-of-sale process on modern smart POS devices.

Payment initiation

A simple API for applications to initiate any type of payment transaction. This includes all types of transaction such as void, refund, pre-authorisation. Also includes the ability to start non-payment flows such as batch-close, reprint receipts and tokenization.

Flow augmentation

Applications that need to augment a payment transaction can be called at any time during an AppFlow process. These applications can be called before, during or after payment and can perform any type of augmentation such as upselling, loyalty, advertising, rewards, gathering feedback.

Payment processing

Our standard interface can used to integrate your payment application to an AppFlow. This means you only need to integrate your payment application once and it will be compatible with all AEVI AppFlow enabled devices.
