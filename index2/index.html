Get started with AEVI AppFlow

Integration with AppFlow follow different paths depending on what type of application you are building. Get started with the AppFlow journey from the choices below and when you are done, make sure to check out the technical details section for further information.

Flow initiation

Client API for applications (typically POS apps) to initiate flows for various functions such as sale, refund and tokenisation.

Flow augmentation

Provides the ability for applications (such as loyalty schemes) to read and augment data at different stages of the flow.

Payment processing

Allows payment applications to integrate with AppFlow and process payment related requests, using any payment method of their choice.
